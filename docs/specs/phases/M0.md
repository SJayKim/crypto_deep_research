## Inherits (from: nothing — first phase)

Greenfield. The repo holds docs + config only (`docs/DESIGN.md`, `CLAUDE.md`,
`.env.example`, `.gitignore`, `.claude/` security hook). No Python, no package, no
scaffold. Nothing to refactor or preserve — M0 is the genuine first code.

**Preconditions:** none.

## Scope

`pyproject.toml` (uv, ruff, mypy, pytest, pre-commit), the package layout (index →
Architecture), all `contracts/` models (index → Shared Contracts, **normative**),
`wiring.py`, the `sources/fixtures/` directory (empty placeholder), and service-URL env
vars added to `.env.example`. **No service logic, no agent, no server.**

## Acceptance criteria (numbered, pass/fail)

1. `uv sync` succeeds; `python -c "import crypto_deep_research.contracts"` works.
2. `uv run ruff check .` and `uv run mypy .` pass clean on `contracts/` + `wiring.py`.
3. `WorkerArtifact` rejects 6 key_points, rejects a >200-char key_point, and rejects
   a >200-char headline (3 deterministic-stub validator unit tests, T7b).
4. `SynthesisReport` round-trips `status` ∈ {ok, partial, failed} with populated
   `dimensions_ok` / `dimensions_unavailable`.
5. `wiring.py` reads the worker URL list + `WORKER_TIMEOUT_S` (default 30) from env;
   a missing var raises a clear error (not a silent default for URLs).
6. No function exceeds 30 lines; no untyped `dict` in contract models (CLAUDE.md).

## Hands off (to: [M1](./M1.md))

M1 builds directly on these M0 outputs:

- **`crypto_deep_research/contracts/` package** — imported by every later service (C5). M1's
  market-worker emits `WorkerArtifact`; the MCP server speaks the `mcp_tools` schemas.
- **`crypto_deep_research/wiring.py`** — env-driven service URLs + `WORKER_TIMEOUT_S`. M1 reads
  the MCP URL from here.
- **`mcp_server/sources/fixtures/`** — empty placeholder; M1 drops `btc_*.json` into it.
- **Tooling green** — `uv sync`, ruff, mypy, pytest all pass. M1 extends the same suite.

**Seam:** contracts are importable and the A2 validators enforce their bounds. M1 may now
write the first service that produces a contract-valid artifact without inventing types
that others must later match.
